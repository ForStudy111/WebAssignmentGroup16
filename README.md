# Counselling Session Booking System – Group 16

A web-based counselling appointment management system developed for **CSE3023 Web-Based Application Development**.

The system allows students to book counselling sessions, counsellors to manage schedules and bookings and administrators to manage counsellor and student information.

---

## Technologies Used

- Java
- Java Servlet
- JSP
- HTML, CSS and JavaScript
- MySQL
- JDBC
- Apache Tomcat
- XAMPP / phpMyAdmin
- Google Calendar API
- NetBeans IDE

---

## Main Functions

### Student
- Log in to the system
- View available counsellor schedules
- Book a counselling session
- Reschedule or cancel pending bookings
- View booking history
- Submit feedback after a completed session
- Update profile information

### Counsellor
- Manage available schedules
- View student booking requests
- Approve, reject or cancel bookings
- Record counselling session notes
- Connect Google Calendar for appointment synchronization

### Administrator
- View and update student information
- Add, edit and delete counsellor accounts
- View and monitor booking records

---

## System Requirements

Before running the system, install:

- JDK
- NetBeans IDE
- Apache Tomcat
- XAMPP
- MySQL Connector/J library
- Web browser such as Google Chrome or Microsoft Edge
